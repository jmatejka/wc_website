# Weight Collector Landing Site

Marketing site for the app, built with React + TypeScript + Vite.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Deploy on Vercel

This project is preconfigured with `vercel.json`.

In Vercel:

- Select this repository.
- Set **Root Directory** to `website`.
- Build command: `npm run build`
- Output directory: `dist`

Then deploy.
